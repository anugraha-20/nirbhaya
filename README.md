### Project Description
Nirbhaya is a women safety web application designed to help users travel safely by providing secure routes, emergency alerts, and instant SOS support.
The system combines map navigation, real-time location sharing, emergency communication, and alert mechanisms into a single platform.

It is designed especially for students, working women, and travelers to feel safe while moving alone.

### The Problem statement
People face safety risks while travelling alone and lack quick emergency support.

### The Solution
üëâ Solution: Nirbhaya provides safe routes, siren alerts, SOS calls, and live location sharing in one click.
